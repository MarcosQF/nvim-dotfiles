# 🌙 Neovim Config - Marcos Queiroz

My personal Neovim setup focused on performance, intuitive keymaps, and a robust terminal workflow.

## 📦 Prerequisites

* **[Neovim](https://neovim.io/)** (0.9.0+)
* **Git**
* A **Nerd Font** (for UI icons)
* **C Compiler** (`gcc` or `clang`) & **Make** (Strictly required by `nvim-treesitter` to build language parsers)

## 🚀 Installation

```bash
git clone https://github.com/MarcosQF/nvim-dotfiles.git ~/.config/nvim
```
